OVERVIEW
This is a simple Python-based Password Generator that creates strong, random, and secure passwords.
The program allows the user to choose the password length and automatically generates a mix of letters, numbers, and special characters.

FEATURES
- Generate strong random passwords
- Customizable password length
- Uses letters, digits, and special symbols
- Beginner-friendly and easy to understand
- No external libraries needed (uses Python's built-in random and string modules)

TECHNOLOGIES USED
- Python 3
- Built-in modules: random, string

HOW TO RUN
1. Install Python 3 on your system.
2. Download or clone the project.
3. Open terminal/cmd in the project folder.
4. Run:
   python password_generator.py

SAMPLE OUTPUT
Enter password length: 12
Generated Password: t@G7m!2zQw#e
